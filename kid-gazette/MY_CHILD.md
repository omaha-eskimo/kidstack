# My Child — Content Profile

Fill this out in plain English. There's no right length — more detail produces
better content. You don't need to answer every question; skip anything that
doesn't fit your kid.

When you're done, open Claude Code in this folder and say:
**"Set up the gazette for [child's name] using MY_CHILD.md."**

Claude will generate the full content bank and configure everything. You won't
need to touch any JSON or code.
